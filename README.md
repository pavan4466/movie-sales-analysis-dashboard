# 🎬 Movie Sales Analysis Dashboard

## 📌 Project Overview

The **Movie Sales Analysis Dashboard** is an interactive Business Intelligence project developed using **Power BI** to analyze movie industry data and uncover meaningful insights related to genres, revenue performance, IMDb ratings, metascores, audience votes, and top-performing directors and movies.

This dashboard transforms raw movie datasets into visually compelling and actionable insights through advanced data visualization techniques, interactive filtering, and storytelling dashboards.

The primary objective of this project is to help users easily identify:

* High-performing movie genres
* Revenue-driving categories
* Audience preferences
* IMDb and Metascore trends
* Top directors and blockbuster movies
* Correlation between ratings and revenue

The dashboard follows a cinematic dark-theme UI to create a professional and engaging user experience.
